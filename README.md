# iPhone 15 Pro Website Clone

A visually immersive clone of Apple’s iPhone 15 Pro landing page, built to replicate the sleek design and interactive experience of the original. This project showcases advanced front-end techniques, 3D modeling, and engaging animations for a premium user experience.

> 🌐 **Live Demo:**  
> [r3f-iphone15-pro-clone.vercel.app](https://r3f-iphone15-pro-clone.vercel.app)

## 🚀 Technologies Used

- **React** – Component-based UI framework
- **Three.js** – Powerful 3D graphics rendering library
- **React Three Fiber** – React renderer for Three.js
- **@react-three/drei** – Helpers for faster 3D development with R3F
- **Tailwind CSS** – Utility-first CSS framework for rapid UI styling
- **GSAP** – Industry-standard JavaScript animation library

Inspired by the [JavaScript Mastery](https://www.jsmastery.pro/) Tutorial.

---

## ✨ Features

### 🎥 Beautiful Subtle Smooth Animations using GSAP

Experience a refined and fluid UI with seamless transitions and animations that enhance the storytelling. GSAP powers all key movements and interactive effects, ensuring a polished, dynamic feel throughout the site.

### 📱 3D Model Rendering with Different Colors and Sizes

Interact with a fully rotatable iPhone 15 Pro model. Users can view it in real-time 3D, switch between multiple colors, and choose different size variations — all rendered using Three.js and React Three Fiber.

### 🎞️ Custom Video Carousel (GSAP-powered)

Engage with a custom-built video carousel that combines scroll-triggered and timeline-based animations. Created with GSAP to deliver a compelling and responsive content display mechanism.

### 🧩 Modular and Reusable Code Architecture

The project is built with scalability in mind. Components are reusable, cleanly separated, and follow best practices in naming, structure, and organization. Utility functions, animation handlers, and constants are abstracted for maximum maintainability.

### 📱 Fully Responsive Design

The layout is mobile-friendly and optimized for various screen sizes, ensuring a consistent user experience across smartphones, tablets, and desktops.

## 🛠️ Getting Started

```bash
# Clone the repo
git clone https://github.com/delafuentej/r3f_iphone15Pro-clone.git
# Navigate into the project
cd r3f_iphone15Pro-clone

# Install dependencies
yarn install

# Start the dev server
yarn dev
```
